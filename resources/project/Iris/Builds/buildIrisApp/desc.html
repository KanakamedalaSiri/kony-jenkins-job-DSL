Guide

This is the home of your App FactoryTM project.
It's defined to contain a collection of jobs and credentials to help you build and test your app.

  • The jobs are predefined and you only need to worry about providing the input parameters you'd like them to use.
  • The credentials are for you to define and will provide your jobs with the access they'll need to integrate with other tools on your behalf.

Jobs

Every App FactoryTM project comes with the following jobs:

  • Build
    • Build a Iris App
    • Define Watchers and Schedulers for building periodically or whenever changes are made to your app.
  • Integrate
    • Export & version a Foundry App
    • Import a Foundry App
    • Publish a Foundry App
  • Test
    • Define Device Pools for testing
    • Test a Iris app on a Device Pool.
    • Define Watchers and Schedulers to trigger tests periodically or whenever changes are made to your test scripts.

Credentials

Every App FactoryTM project will require you to define credentials of the following types:

  • Volt MX Foundry User & Password
  • Volt MX Foundry App Configuration
  • SCM User & Password or User & Personal Access Token
  • Android Signing Secrets
  • iOS Signing Secrets
